DD TRAINER - SIMULADOR DD-602 / DD-1000
========================================

ABRIR EL SIMULADOR
------------------
1. Descomprime el archivo ZIP completo.
2. Entra en la carpeta DD-Trainer-Portable.
3. Haz doble clic en index.html.
4. Pulsa POWER y luego sube MAIN VOLUME para activar los sonidos.

No necesita conexión a Internet ni instalación.

CONTROLES DE LA CONSOLA
-----------------------
- DATA: arrastra la rueda hacia la izquierda o derecha. También acepta la
  rueda del mouse y las flechas del teclado cuando tiene el foco.
- TEMPO / TAP: púlsalo cuatro veces siguiendo el pulso para detectar el BPM.
  También puedes ajustar el valor exacto con DATA.
- PAGE - / +: cambia de página en SONG, VOICE, UTILITY y RECORD. El visor
  central muestra la página actual y el total (por ejemplo, 2/8).
- RECORD: elige una canción de usuario 51-54, avanza con PAGE + hasta
  RECORD READY y pulsa START / STOP. Toca los pads y vuelve a pulsar
  START / STOP para guardar. Púlsalo otra vez para escuchar lo grabado.
- VOLUMEN DE UN INSTRUMENTO: golpea el pad que quieres editar, pulsa VOICE,
  avanza dos veces con PAGE + hasta VOLUME / PAN y usa DATA sobre >Vol.
  Cada pad conserva su propio volumen y paneo dentro de cada kit.

VERSIÓN 1.4
-----------
La rueda DATA ahora es más grande, muestra claramente DATA, − y +, y su parte
móvil gira al arrastrar, usar la rueda del mouse o las flechas. El curso fue
reorganizado en 8 prácticas esenciales: encendido, uso de DATA, kits, volumen
individual, sensibilidad y curva, metrónomo, asignación/guardado y grabación.
El progreso usa una versión nueva, comienza en 0/8 y puede reiniciarse desde
el botón Reiniciar progreso. Los pasos solo avanzan al realizar la acción.

VERSIÓN 1.3
-----------
Incluye un REAR PANEL reconstruido según el manual, con MIDI DIN de 5 pines,
entradas de disparo, AUX IN, salidas AUX OUT y DC IN 9V diferenciados. También
mejora la alineación de ACCOMP/PAGE y conserva el volumen y paneo independientes
por pad y kit, TEMPO/TAP, DATA, PAGE y RECORD corregidos.

CONTENIDO
---------
- index.html: abre el simulador portátil.
- app.js y app.css: archivos necesarios para ejecutarlo.
- codigo-fuente: versión editable en React y TypeScript.
- manual: PDF original DD-602 / DD-1000 utilizado como referencia.

EDITAR EL CÓDIGO FUENTE
-----------------------
Requisitos: Node.js 22 o posterior.

1. Abre una terminal dentro de codigo-fuente.
2. Ejecuta: npm install
3. Ejecuta: npm run dev

Para reconstruir la versión portátil:
npm run build:portable

IMPORTANTE
----------
El audio es una recreación didáctica generada por el navegador. No es una
emulación exacta de los sonidos originales del hardware.
